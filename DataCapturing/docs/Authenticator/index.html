Protocol Authenticator 

public protocol Authenticator

An Authenticator provides functionality to authenticate this app on a servoer for receiving the captured data.

Types Conforming to Authenticator

CredentialsAuthenticator

An authenticator using a combination of username and password to authenticate against a Cyface data collector server.

StaticAuthenticator

A StaticAuthenticator always does authentication using the same Java Web Token (JWT). This token is provided as a fixed value and never changes. The creator of an object of this class needs to make sure, that the token is valid.

Runs the authentication and calls the appropriate function onSuccess or onFailure when finished.

Parameters

A closure called and supplied with the resulting authentication token, when authentication was successful.
