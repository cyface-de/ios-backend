CredentialsAuthenticator

public class CredentialsAuthenticator: Authenticator

An authenticator using a combination of username and password to authenticate against a Cyface data collector server.

After creation of an instance of this class you need to provide a username and a password, before calling authenticate(). If nousernameorpasswordis available the function will call its failure handler with theServerConnectionError.Category.notAuthenticated`.
