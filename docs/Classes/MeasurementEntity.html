MeasurementEntity

public class MeasurementEntity

Represents one measurement carried out by the Cyface SDK.

A measurement is a track of geo locations and associated accelerations.

Remark

DO NOT confuse this class with CoreData generated model object MeasurementMO. Since the model object is not thread safe you should use an instance of this class if you hand data between processes.
